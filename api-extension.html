SZ10G21 - API Extension

Base URL:

https://ecs.soton.ac.uk

Endpoints:

Resource URI Methods Accept Types Description
Modules /module/ GET POST OPTIONS JSON XML HTML Retrieve a list of all modules, create a new module, or check allowed methods.
Specific Module /module/{code} GET PUT PATCH DELETE OPTIONS JSON XML HTML Retrieve, update, delete a specific module, or check allowed methods.
Deliveries /module/{code}/delivery/{crn} GET POST OPTIONS JSON XML HTML Retrieve details of a specific delivery, create a new delivery for a module, or check allowed methods.
Assignments /module/{code}/delivery/{crn}/assignment/{arn} GET POST OPTIONS JSON XML HTML Retrieve details of a specific assignment, create a new assignment within a delivery, or check allowed methods.
Students /student/{username} GET POST PUT PATCH DELETE OPTIONS JSON XML HTML Retrieve, create, update, delete a student record, or check allowed methods.
Submissions /module/{code}/delivery/{crn}/assignment/{arn}/submission/{username}/{srn} GET POST PUT PATCH DELETE OPTIONS JSON XML HTML Manage submissions for a specific assignment by a student, or check allowed methods.
File /module/{code}/delivery/{crn}/assignment/{arn}/submission/{username}/{srn}/{fileName} GET OPTIONS JSON XML PDF Access the submitted file or check allowed methods.

Breakdown:

Segment Description
/module/{code} Identifies a specific module using its unique code (e.g., COMP3227).
/delivery/{crn} Specifies the delivery of the module identified by its Course Reference Number (CRN) (e.g., 050198XML).
/assignment/{arn} Denotes a specific assignment within the delivery (e.g., 1 for the first assignment).
/submission/{username}/{srn} Represents a specific submission made by a student identified by their username and submission number.
/{fileName} Directly accesses the submitted file (e.g., Brief.pdf).

Example:

https://ecs.soton.ac.uk/module/COMP3227/delivery/050198XML/assignment/1/submission/sz10g21/1/Brief.pdf

Evaluated:

  • module/COMP3227: Identifies the COMP3227 module.
  • delivery/050198XML: Specifies the delivery identified by CRN 050198XML.
  • assignment/1: Denotes the first assignment of the delivery.
  • submission/sz10g21/1: Represents the first submission made by student sz10g21.
  • Brief.pdf: The actual file submitted.

Thus, this URL allows you to access the specific submission file made by a student for a particular assignment in a module's delivery as per URI:

https://ecs.soton.ac.uk/module/{code}/delivery/{crn}/assignment/{arn}/submission/{username}/{srn}/{fileName}

Each response includes a _links section containing relevant hypermedia links. These links provide URLs to related resources and available actions.

Semantics:

Placeholder Description
{code} Module Code (e.g., COMP3227)
{crn} Course Reference Number identifying the delivery (e.g., 050198XML)
{arn} Assignment Number (e.g., 1 for the first assignment)
{srn} Submission Number (e.g., 1 for the first submission)
{username} Unique identifier for a student (e.g., sz10g21)
{fileName} Name of the submitted file (e.g., Brief.pdf)
